📋 Einrichtungsanleitung

  1. Scraper starten: In der .env auf dem Server ENABLE_ANDROID_SCRAPER=true setzen, dann
    docker compose up -d android-scraper.
  2. Einmalig bei Google anmelden: Oben auf Verbinden klicken. Im sichtbaren Chromium-Fenster mit dem Google-Konto einloggen, das mit deinen Android-Geräten verknüpft ist. Chrome Sync aktivieren.
  3. Fertig – läuft automatisch: Der Container scrapet danach alle 5 Minuten Google MyActivity. Die Session übersteht Container-Neustarts (persistentes Volume chrome-profile).
  4. Browser-Extension (optional): Für das Erfassen des PC-Browser-Verlaufs das vorkonfigurierte Paket oben herunterladen und unter chrome://extensions → Entwicklermodus → „Entpackte Extension laden" installieren. Gateway URL und Token sind vorausgefüllt.
